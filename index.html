Highlight Translate - Privacy Policy

Last updated: March 24, 2026

Overview

Highlight Translate is a Chrome extension that allows users to highlight text on any webpage and translate it to Chinese. We are committed to protecting your privacy.

Data We Collect

Website Content: The extension reads the text you select (highlight) on a webpage to send it to Google Translate for translation. This text is sent directly to Google's translation API and is not stored, logged, or transmitted to any other server.

Data We Do NOT Collect

  • We do not collect personally identifiable information
  • We do not collect health, financial, or payment information
  • We do not collect authentication information (passwords, credentials)
  • We do not collect personal communications (emails, messages)
  • We do not collect location data or IP addresses
  • We do not collect web browsing history
  • We do not monitor user activity (clicks, keystrokes)

Third-Party Services

Selected text is sent to Google Translate API (translate.googleapis.com) for translation. Google's privacy policy applies: https://policies.google.com/privacy

Data Storage

Highlight Translate does not store any user data. All translations are performed in real-time and no text is saved locally or remotely.

Permissions

  • activeTab: To read the text you select on the current page for translation
  • scripting: To inject the translate icon and popup UI into web pages
  • translate.googleapis.com: To send selected text for translation

No Remote Code

This extension does not load or execute any remote code. All JavaScript is bundled within the extension package.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.
