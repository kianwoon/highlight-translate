Privacy Policy for Highlight Translate

Last updated: April 7, 2026

Highlight Translate ("the Extension") is committed to protecting your privacy. This policy describes what data the Extension collects, how it is used, stored, and shared.

1. Data We Collect

The Extension collects only the minimum data necessary to provide its core functionality:

Data TypeDescriptionPurpose
User Preferences Selected AI provider, model name, custom prompt templates, and custom API endpoint URL Remember your settings between sessions
API Keys API key for your chosen AI provider (e.g., OpenAI, Anthropic, Google Gemini) Authenticate requests to the AI service on your behalf
Selected Text Text you highlight on a webpage when triggering a translation, improvement, reply, or summary action Process your request via the selected AI provider

We do not collect: browsing history, personal identification information, cookies, analytics data, or any data unrelated to the Extension's functionality.

2. How We Use Your Data

  • User Preferences & API Keys — Stored locally in your browser using the Chrome Storage API (chrome.storage.local). These are used solely to persist your configuration across browser sessions.
  • Selected Text — Sent directly from your browser to your chosen AI provider's API for processing. The text is used only to generate the requested output (translation, improvement, reply, or summary).

3. Data Storage

  • All user preferences and API keys are stored locally on your device using Chrome's built-in local storage (chrome.storage.local).
  • We do not operate any servers, databases, or cloud services that store your data.
  • Data persists only on your device until you clear Extension data or uninstall the Extension.

4. Data Sharing & Third Parties

The Extension shares data only with the AI provider you configure. No data is sent to any other party.

Third PartyData SharedPurposeProvider Privacy Policy
Google (Gemini API) Highlighted text and your custom prompt AI-powered translation, improvement, reply, and summary generation https://policies.google.com/privacy
OpenAI (API) Highlighted text and your custom prompt AI-powered translation, improvement, reply, and summary generation https://openai.com/privacy
Anthropic (API) Highlighted text and your custom prompt AI-powered translation, improvement, reply, and summary generation https://www.anthropic.com/privacy
Custom Endpoint Highlighted text and your custom prompt AI-powered translation, improvement, reply, and summary generation via user-configured endpoint Determined by the user's chosen endpoint provider
Google Translate Highlighted text Free translation (fallback / default provider) https://policies.google.com/privacy

Important: When you configure a custom API endpoint, the highlighted text will be sent to that endpoint. You are responsible for reviewing the privacy practices of your chosen endpoint provider.

5. Data Retention

  • Local data (preferences, API keys) is retained until you clear Extension data or uninstall the Extension.
  • AI provider data retention is governed by each provider's respective privacy policy (linked above). The Extension itself has no control over how long these providers retain API request data.

6. User Rights & Control

  • You may view, modify, or delete your stored preferences and API keys at any time via the Extension's Options page.
  • Uninstalling the Extension removes all locally stored data.
  • You may choose which AI provider to use, thereby controlling which third party receives your text data.

7. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect data from children under 13.

8. Security

  • API keys are stored locally using Chrome's secure storage mechanism and are never transmitted except to authenticate with your chosen AI provider over HTTPS.
  • All API communications use encrypted HTTPS connections.
  • No data is transmitted to any server operated by the Extension developer.

9. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date above. Continued use of the Extension after changes constitutes acceptance of the updated policy.
